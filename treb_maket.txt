Требования (формальное описание):

Макет пользовательского интерфейса игры-тренажера
Угадай фигуру по контуру

Для работы макета необходимы:
Аппаратные средства:
любое устройство, имеющее браузер, поддерживающий HTML и JavaScript.

Программное обеспечение: 
браузер, поддерживающий HTML и JavaScript.

Требования к разработке:

ПО для разработки:
текстовый редактор, 
браузер.  

Языки, используемые при разработке:
HTML, JavaScript

Для разработки макета:

Создать html-страницы index.html, game.html, ans.html, user.html, settings.html, empty.html

index:
краткая инструкция - текст;
кнопка Начать игру - переход на game;
кнопка Результаты - переход на user;
кнопка Настройки - переход на settings;
кнопка Выход - переход на empty.

game:
кнопка Угадал(а) - переход на ans;
кнопка Открыть фигуру - переход на empty;
ссылка На главную - переход на index.

ans:
форма содержит группу кнопок с зпвисимой фиксацией, кнопки Отправмть и Отмена, список возможных фигур (не является диалоговым элементом);
элементы формы разместить в таблице;
кнопка Отпрпвить - скрипт проверяет, выбрана ли 1 кнопка, если выбрана - переход на empty, иначе - сообщение "Фигура не выбрана";
 На главную - переход на index.

user:
таблица с результатами сеансов игры;
кнопка Ok - переход на index;
ссылка На главную - переход на index.

settings:
форма содержит раскрывающийся список уровней сложности, кнопку Загрузить, кнопки Отправмть и Отмена;
элементы формы разместить в таблице;
кнопка Загрузить - переход на empty;
раскрывающийся список для выбора уровея сложности, по умолчанию выбран уровень 1;
кнопка Отправить - переход на empty;
кнопка Отмена - присвоить полям формы значения по умолчанию;
ссылка На главную - переход на index.

empty:
Заглушка - текст;
кнопка Назад - переход на предыдущую страницу;
ссылка На главную - переход на index.
